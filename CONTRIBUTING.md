# 贡献指南

欢迎贡献代码！

---

## 贡献流程

1. **Fork** 项目到你的账户
2. 创建功能分支：`git checkout -b feature/xxx`
3. 提交代码：`git commit -m 'feat: 描述'`
4. 推送分支：`git push origin feature/xxx`
5. 提交 **Pull Request**

---

## 提交规范

| 类型 | 说明 |
|------|------|
| `feat` | 新功能 |
| `fix` | Bug 修复 |
| `docs` | 文档更新 |
| `style` | 代码格式 |
| `refactor` | 重构 |
| `chore` | 构建/工具 |

---

## 注意事项

- 遵循 Kotlin 编码规范
- 确保项目能正常构建
- 不要提交敏感信息（密钥、密码等）
- 本项目使用 MIT 许可证
